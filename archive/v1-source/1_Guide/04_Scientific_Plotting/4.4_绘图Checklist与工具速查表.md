# 4.4 绘图 Checklist 与工具速查表

## 绘图 Checklist：提交前的自查清单

在将图表插入论文之前，请逐一检查以下清单：

| 检查项 | 是否通过 |
|--------|---------|
| 图表是否使用矢量格式（PDF/EPS/SVG）导出？ | ☐ |
| 图表中的字体在 LaTeX 编译后清晰可读（不小于 8pt）？ | ☐ |
| 图注（Caption）是否完整，让读者独立理解图表内容？ | ☐ |
| 坐标轴标签（Axis Labels）完整，包含单位？ | ☐ |
| 图例（Legend）是否清晰，与图中元素一一对应？ | ☐ |
| 是否同时使用了颜色+线型/标记进行双重编码（色盲友好）？ | ☐ |
| Y 轴的起点和范围是否合理，不误导读者？ | ☐ |
| 图中的模块名称/符号是否与正文一致？ | ☐ |
| 你的方法（Ours）是否在视觉上适当高亮？ | ☐ |
| 图表是否使用代码生成（可重复性）？ | ☐ |

## 论文绘图工具速查表

最后，总结一下不同类型图表的推荐工具选择：

| 图表类型 | 首选工具 | 备选工具 | 说明 |
|---------|---------|---------|------|
| Motivated Example Figure | PPT / Figma / OmniGraffle | draw.io | 包含文字、图标、代码片段等混合元素，需要灵活排版 |
| Solution Overview Figure | draw.io / PPT | Figma / TikZ | 流程图和架构图，需要清晰的模块划分和连线 |
| Experimental Results Figure | Matplotlib + Seaborn | TikZ / PGFPlots | 必须使用代码生成，保证可重复性 |
| 数学公式/定理图示 | TikZ | Matplotlib | 需要与 LaTeX 完美融合 |
